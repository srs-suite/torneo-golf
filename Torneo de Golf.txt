__________________________________

Ver los cambios: Nginx sirve el frontend desde /home/retailso/torneogolf.retailsolutionstimetracker.com, no desde torneogolf-source. Por eso había que copiar ahí el dist después del build. De ahora en adelante, cada vez que hagas npm run build en torneogolf-source/frontend, hay que volver a copiar el contenido de dist a esa carpeta.

ESTO ES LO IMPORTANTE <------
Para la próxima vez que actualices el frontend en el VPS:

cd /home/retailso/torneogolf-source/frontend
npm run build
cp -rf dist/* /home/retailso/torneogolf.retailsolutionstimetracker.com/
pm2 restart teetracker-backend
